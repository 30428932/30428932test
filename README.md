## 🚀 快速部署

### 方法：GitHub Pages（免费推荐）

1. **创建GitHub仓库**
   ```bash
   git init
   git add .
   git commit -m "添加武当山旅游规划网站"
   ```

2. **推送到GitHub**
   ```bash
   git remote add origin https://github.com/你的用户名/wudang-travel.git
   git branch -M main
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 `main` branch
   - 点击 Save

4. **访问地址**
   `https://你的用户名.github.io/wudang-travel/`
